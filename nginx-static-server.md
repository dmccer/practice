# nginx 静态服务器配置

## 需求介绍

1. 能正确返回静态资源
2. 设置静态资源的过期时间
3. 设置静态资源特定域名
4. 设置静态资源存放的根目录
5. 开启 gzip，并调整压缩参数
6. 负载均衡
7. 设置请求错误页面
